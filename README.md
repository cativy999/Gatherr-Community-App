# Gatherr Community App

A community platform for church groups to connect, 
organize events, and manage group activities together.

## Live App
🔗 [gatherr-one.vercel.app](https://gatherr-one.vercel.app)

## About
Designed and built solo using an AI-native workflow —
from Figma and Lovable for design, to Claude and Cursor 
for development. Shipped in under two weeks.

## Tech Stack
- React + TypeScript
- Tailwind CSS + Shadcn UI
- Supabase (auth + database)
- Vite
- Deployed on Vercel

## Features
- User onboarding and profiles
- Community event management
- Member connection tools
